This project is a simple command-line To-Do List application built in Python. It allows users to manage tasks through a menu interface: viewing current tasks, adding new ones, removing completed items, and saving the list for future use.
The project was created to practice and demonstrate core Python fundamentals, including working with lists, functions, loops, conditional logic, and basic file operations. Tasks are stored in a text file, allowing them to persist between program runs. When the program starts, it automatically loads previously saved tasks.
